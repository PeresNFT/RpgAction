# Imagens de UI

Coloque as imagens de interface do usuário nesta pasta.

## Nomenclatura

- `Descanso.png` - Para o botão de descanso

## Formato Recomendado

- **Formato**: PNG (com transparência) ou JPG
- **Tamanho**: 200x200px a 400x400px (será redimensionado automaticamente para w-24 h-24 - 96x96px)
- **Background**: Transparente (preferencial) ou fundo escuro
- **Estilo**: Representativo da ação (descanso, relaxamento, etc.)

## Tamanho Padrão

As imagens serão exibidas com:
- **Largura**: 96px (w-24)
- **Altura**: 96px (h-24)
- **Object-fit**: contain (mantém proporções)
