# Imagens de Monstros

Coloque as imagens dos monstros nesta pasta.

## Nomenclatura

As imagens devem seguir os seguintes nomes (baseados no nome do monstro):

- `goblin.png` - Para "Goblin Raivoso"
- `wolf.png` - Para "Lobo Enfurecido"
- `orc.png` - Para "Orc Esmagador"
- `shadow.png` - Para "Sombra do Caos"
- `eye.png` - Para "Olho da Verdade"
- `demon.png` - Para "Demônio Sussurrante"
- `dragon.png` - Para "Dragão de Cristal"
- `phoenix.png` - Para "Fênix Imortal"
- `titan.png` - Para "Titã de Pedra"
- `vampire.png` - Para "Vampiro Ancestral"
- `lich.png` - Para "Lich Sombrio"
- `kraken.png` - Para "Kraken Abissal"
- `giant.png` - Para "Gigante de Gelo"
- `elemental.png` - Para "Elemental de Fogo"
- `necromancer.png` - Para "Necromante Maldito"
- `mermaid.png` - Para "Sereia Traiçoeira"
- `minotaur.png` - Para "Minotauro Lendário"
- `hydra.png` - Para "Hidra de Nove Cabeças"
- `fallen_angel.png` - Para "Anjo Caído"
- `ancient_god.png` - Para "Deus Antigo"
- `void_creature.png` - Para "Criatura do Vazio"
- `war_lord.png` - Para "Senhor da Guerra"
- `default.png` - Imagem padrão caso não encontre a específica

## Formato Recomendado

- **Formato**: PNG (com transparência) ou JPG
- **Tamanho**: 200x200px a 400x400px (será redimensionado automaticamente)
- **Background**: Transparente (preferencial) ou fundo escuro

## Fallback

Se uma imagem não for encontrada, o sistema usará o ícone emoji do monstro como fallback.
