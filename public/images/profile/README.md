# Imagens de Perfil

Coloque as imagens de perfil exclusivas nesta pasta.

## Nomenclatura

As imagens devem seguir os seguintes nomes:

- `profile1.png` - Foto de perfil exclusiva #1
- `profile2.png` - Foto de perfil exclusiva #2
- `profile3.png` - Foto de perfil exclusiva #3
- `profile4.png` - Foto de perfil exclusiva #4

## Formato Recomendado

- **Formato**: PNG (com transparência) ou JPG
- **Tamanho**: 200x200px a 400x400px (será redimensionado automaticamente)
- **Background**: Transparente (preferencial) ou fundo escuro
- **Estilo**: Representativo de avatar/perfil do jogador

## Uso

Estas imagens são vendidas na loja NPC por diamantes e podem ser selecionadas como foto de perfil do jogador.
