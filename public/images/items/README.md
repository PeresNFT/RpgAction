# Imagens de Itens

Coloque as imagens dos itens do jogo nesta pasta.

## Nomenclatura

As imagens devem seguir os seguintes nomes (baseados no ID do item):

### Poções (Consumáveis)
- `health_potion.png` - Poção de Vida ✅ (já existe)
- `mana_potion.png` - Poção de Mana ✅ (já existe)

### Armas (Weapons)
- `sword_basic.png` - Espada Básica
- `bow_basic.png` - Arco Básico
- `staff_basic.png` - Cajado Básico

### Armaduras (Armor)
- `leather_armor.png` - Armadura de Couro

### Materiais (Materials)
- `herb.png` - Erva Medicinal
- `iron_ore.png` - Minério de Ferro

### Recursos de Coleta - Madeira (Woodcutting)
- `wood.png` - Madeira
- `oak_wood.png` - Madeira de Carvalho
- `pine_wood.png` - Madeira de Pinho

### Recursos de Coleta - Pesca (Fishing)
- `fish.png` - Peixe
- `salmon.png` - Salmão
- `tuna.png` - Atum

### Recursos de Coleta - Mineração (Mining)
- `copper_ore.png` - Minério de Cobre
- `gold_ore.png` - Minério de Ouro

### Recursos de Coleta - Agricultura (Farming)
- `wheat.png` - Trigo
- `corn.png` - Milho

## Formato Recomendado

- **Formato**: PNG (com transparência) ou JPG
- **Tamanho**: 200x200px a 400x400px (será redimensionado automaticamente)
- **Background**: Transparente (preferencial) ou fundo escuro
- **Estilo**: Representativo do item

## Fallback

Se uma imagem não for encontrada, o sistema usará o ícone emoji do item como fallback automaticamente.

## Status

- ✅ = Imagem já existe
- ⬜ = Imagem ainda não foi criada
